🚀 사용 방법
1
비밀번호 설정
앱 첫 실행 시 4자리 PIN 설정. 팀원 모두 같은 번호 사용. 잠금 화면 하단 비밀번호 변경으로 언제든 변경 가능.
2
이달 총원 입력
매달 1일, 화면 하단 이달 인원 변동 등록에서 기준 총원 입력. 신규·복관·휴·퇴관 생길 때마다 업데이트.
3
매일 업무일지 작성 후 저장
부별 출석 입력 → 결석 자동계산. 내용 입력 후 저장하기 누르면 팀원들에게 알림 자동 전송!
4
📅 캘린더로 날짜 이동
헤더 우측 📅 버튼을 누르면 달력이 펼쳐져요. 원하는 날짜 탭 → 그날 업무일지로 바로 이동. 데이터 저장된 날은 파란색으로 표시, 출석 인원도 바로 확인!
5
🔔 실시간 알림
팀원이 저장하면 즉시 팝업 알림 표시. 알림 탭하면 해당 날짜 업무일지로 바로 이동. 🔔 버튼에서 전체 알림 목록 확인.
💡 저장 시 팀원들 폰에 자동 알림이 전송됩니다!
